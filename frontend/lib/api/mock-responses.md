# 医疗档案系统 - 默认演示内容

## 系统介绍

欢迎使用**医疗档案智能问答系统**！

本系统整合了多种医疗档案资料，包括：
- **中医诊疗指南** - 提供规范化的诊疗方案
- **针灸穴位详细图谱** - 精准定位和应用指导
- **中医诊断规范** - 标准化诊断流程
- **中药功效图谱** - 全面的中药材功效信息

---

## 核心功能特性

### 1. 多维度知识检索
系统采用**知识图谱 + 向量检索 + 文档检索**三层架构：
- **知识图谱推理**：基于 Neo4j 构建医疗实体关系网络
- **语义向量检索**：使用 Milvus 进行高效相似度搜索
- **文档精准定位**：MongoDB 存储完整文档结构

### 2. 图文并茂的输出格式
答案自动包含：
- **引用标注**：每个信息点都有明确来源 [1][2][3]
- **图片展示**：自动提取相关插图，支持点击放大查看
- **PDF 跳转**：点击引用可直接跳转到原始文档页面

### 3. 智能推荐问题
基于当前对话上下文，系统会推荐：
- 深入探讨的相关问题
- 横向拓展的关联主题
- 实践应用的案例询问

---

## 示例问答

**问：如何治疗失眠？**

失眠的中医治疗主要从**辨证论治**出发，常见证型包括心脾两虚、阴虚火旺等。

### 治疗方案

#### 1. 针灸治疗
主要穴位：
- **神门穴**（HT7）- 养心安神[image:1]
- **三阴交**（SP6）- 健脾益气，宁心安神
- **百会穴**（GV20）- 醒神开窍

#### 2. 中药调理
推荐方剂：
- **酸枣仁汤** - 适用于心脾两虚型
- **黄连阿胶汤** - 适用于阴虚火旺型

#### 3. 生活调理
建议：
- 规律作息，晚上11点前入睡
- 睡前避免剧烈运动和刺激性饮食
- 保持卧室环境安静舒适

要使用完整功能，请：

3. 重启前端开发服务器
